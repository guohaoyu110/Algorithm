# 38. 二叉树的深度（树）

题目描述
----

输入一棵二叉树，求该树的深度。从根结点到叶结点依次经过的结点（含根、叶结点）形成树的一条路径，最长路径的长度为树的深度。

```python
# -*- coding:utf-8 -*-
class TreeNode:
    def __init__(self, x):
        self.val = x
        self.left = None
        self.right = None
class Solution:
    # 还需要理解一下递归的用法！！！
    def TreeDepth(self, pRoot):
        # write code here
        if pRoot == None:
            return 0
        count = max(self.TreeDepth(pRoot.left), self.TreeDepth(pRoot.right))+1
        return count
```
